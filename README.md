# API PHP MVC

Ce projet est une API RESTful développée en PHP suivant le modèle d'architecture MVC (Modèle-Vue-Contrôleur). Elle permet de gérer des opérations CRUD (Create, Read, Update, Delete) sur des ressources spécifiques.

## Fonctionnalités

- **Création** : Ajouter de nouvelles ressources à l'API.
- **Lecture** : Récupérer des informations sur les ressources existantes.
- **Mise à jour** : Modifier les ressources existantes.
- **Suppression** : Supprimer des ressources de l'API.

## Prérequis

- **Serveur Web** : Apache ou Nginx recommandé.
- **PHP** : Version 7.2 ou supérieure.
- **Base de données** : MySQL ou MariaDB.
